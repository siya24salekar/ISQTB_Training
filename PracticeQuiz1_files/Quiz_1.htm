ISEB / ISTQB Foundation in Software Testing Training Course

 

Practice Quiz 1

1) An Incident Report should include all but which of the following? Show Answer

The correct Answer is 'C'
a) Date of Issue
b) Severity of the impact on the system
c) Name of developer who made the mistake
d) Urgency/priority to fix

2) Capture and replay tools are useful for which type of testing? Show Answer

The correct Answer is 'A'
a) Regression Testing
b) Integration Testing
c) User Acceptance Testing
d) Unit Testing

3) The Planning stage of a review includes all but which of the following? Show Answer

The correct Answer is 'C'
a) Selecting the personnel
 b) Allocating roles
c) Fixing defects
 d) Defining the entry and exit criteria

4) The Follow-up stage of a review includes all but which of the following? Show Answer

The correct Answer is 'C'
a) Checking that defects have been addressed
b) Gathering metrics
c) Checking on Entry Criteria
d) Checking of Exit Criteria

5) Typical defects discovered by static analysis tools include all but which of the following? Show Answer

The correct Answer is 'C'
a) Variables that are never used
b) Programming standards violations
c) Reachable code
d) Syntax violations of code and software models

6) Which one of the following does not apply to potential environmental failure conditions? Show Answer

The correct Answer is 'C'
a) Radiation
b) Magnetism
c) Pollution
d) Electronic fields

7) With the help of ______, it is possible to measure the quality of software in terms of defects found. Show Answer

The correct Answer is 'B'
a) Defects
b) Testing
c) Reviewing
d) Learning

8) Which of the following is not a non-functional characteristic? Show Answer

The correct Answer is 'B'
a) Reliability
b) Testability
c) Usability
d) Portability

9) Deciding how much testing is enough, should take into account of the level of what? Show Answer

The correct Answer is 'C'
a) Testing
b) Time
c) Risk
d) Resources

10) The General Testing Principles include all but which of the following? Show Answer

The correct Answer is 'B'
a) Early Testing
b) Absence of Errors Paradox
c) Defect Clustering
d) Exhaustive testing is impossible

11) A common type of V-model uses all but which of the following four test levels? Show Answer

The correct Answer is 'C'
a) Integration Testing
b) Acceptance Testing
c) Maintenance Testing
d) System Testing

12) Which of the following is not considered to be a software work product? Show Answer

The correct Answer is 'A'
a) CMMI
b) Business Scenarios
c) Use Cases
d) Design Documentation

13) Development testing may include all but which of the following: Show Answer

The correct Answer is 'C'
a) Component Testing
b) Integration Testing
c) Acceptance Testing
d) System Testing

14) Which of the following is what a human can make? Show Answer

The correct Answer is 'C'
a) A bug
b) A defect
c) An error
d) A fault

15) Which of the following means the same as an error? Show Answer

The correct Answer is 'C'
a) A bug
b) A defect
c) A mistake
d) An error

16) An informal review includes all but which of the following characteristics? Show Answer

The correct Answer is 'C'
a) Optionally may be documented
b) May vary in usefulness depending on the reviewer
c) Includes metrics
d) Inexpensive way to get some benefit

17) Looking for failures in a system requires all but which of the following? Show Answer

The correct Answer is 'B'
a) Curiosity
b) Attention to managers
c) Professional pessimism
d) A critical eye

18) Defects found and fixed during testing will save time and money when? Show Answer

The correct Answer is 'B'
a) Never
b) Later
c) Sometimes
d) Always

19) Which of the following is not considered to be under the ISTQB code of ethics? Show Answer

The correct Answer is 'C'
a) Client and Employer
b) Judgment
c) Determination
d) Profession

20) Testing can give confidence in the quality of the software if it finds which? Show Answer

The correct Answer is 'B'
a) Lots of defects
b) Few or no defects
c) Too many defects to count
d) Less defects than previous projects

21) What does COTS represent? Show Answer

The correct Answer is 'C'
a) Commercial Off The Shelves
b) Commercial Out The Shop
c) Commercial Off The Shelf
d) Commercial On The Shelf

22) The General Testing Principles include all but which of the following? Show Answer

The correct Answer is 'D'
a) Testing shows presence of defects
b) Defect clustering
c) Pesticide Paradox
d) Exhaustive testing is possible

23) Different development life cycle models need different what, to testing? Show Answer

The correct Answer is 'D'
a) Managers
b) Testers
c) Developers
d) Approaches

24) Operational Acceptance testing includes all but which of the following? Show Answer

The correct Answer is 'C'
a) Disaster recovery
b) User management
c) Periodic migration
d) Maintenance tasks

25) Regulation acceptance testing is performed against any regulations that must be what? Show Answer

The correct Answer is 'C'
a) Understood
b) Read
c) Adhered to
d) Followed

26) Site acceptance testing is another term for what? Show Answer

The correct Answer is 'C'
a) Operational Acceptance testing
b) User Acceptance testing
c) Alpha & Beta testing
d) Regression testing

27) Which tests are based on functions and features? Show Answer

The correct Answer is 'A'
a) Functional tests
b) Acceptance tests
c) Regression tests
d) User Acceptance tests

28) Lessons learned from previous projects, is an aspect of what? Show Answer

The correct Answer is 'B'
a) Quality Insurance
b) Quality Assurance
c) Assurance Quality
d) Risk Assurance

29) Benefits of reviews include all but which of the following? Show Answer

The correct Answer is 'C'
a) Early defect detection and correction
b) Development productivity improvements
c) Increased development timescales
d) Fewer defects

30) A Walkthrough review includes all but which of the following characteristics? Show Answer

The correct Answer is 'B'
a) Open-ended sessions
b) Pre-meeting preparation
c) Scenarios or dry runs
d) May vary in practice from quite informal to very formal

31) Which of the following is not a characteristic of an Inspection review? Show Answer

The correct Answer is 'A'
a) Inexpensive way to get some benefit
b) Defined roles
c) Usually conducted as a peer examination
d) Includes metrics gathering

32) The Kick-off stage of a review includes all but which of the following? Show Answer

The correct Answer is 'A'
a) Gathering metrics
b) Distributing documents
c) Explaining the objectives
d) Checking entry criteria

33) The Fundamental Test Process includes all but which of the following activities? Show Answer

The correct Answer is 'D'
a) Planning and control
 b) Analysis and design
c) Evaluating exit criteria and reporting
d) Testing and execution

34) An informal review includes all but which of the following characteristics? Show Answer

The correct Answer is 'D'
a) Optionally may be documented
b) May vary in usefulness depending on the reviewer
c) Inexpensive way to get some benefit
d) Led by author

35) Which of the following is not a success factor for a review? Show Answer

The correct Answer is 'C'
a) Each review has clear predefined objectives
b) The right people for the review objectives are involved
c) People issues and psychological aspects are postponed *
d) Defects found are welcomed and expressed objectively

36) Which of the following is not relate to a test execution schedule? Show Answer

The correct Answer is 'C'
a) Regression tests
b) Prioritization
c) Acceptance tests
d) Technical and logical dependencies

37) The main purpose of a Technical review includes all but which of the following? Show Answer

The correct Answer is 'B'
a) Checking conformance to specifications and standards
b) Learning
c) Evaluate alternatives
d) Find defects

38) What are used to decide where to start testing and where to test more? Show Answer

The correct Answer is 'B'
a) Project plans
b) Risks
c) Management suggestions
d) Test results

39) The Fundamental Test Process includes all but which of the following activities? Show Answer

The correct Answer is 'C'
a) Planning and control
 b) Analysis and design
c) Testing and execution
d) Evaluating exit criteria and reporting

40) An Inspection review includes all but which of the following? Show Answer

The correct Answer is 'A'
a) Open-ended sessions
b) Includes metrics
c) Pre-meeting preparation
d) Formal follow-up process
