Practice Quiz 5

1) Defects in software may result in what? Show Answer

The correct Answer is 'C'
a) A mistake
b) A defect
c) A failure
d) A fault

2) Test activities exist before and after what? Show Answer

The correct Answer is 'A'
a) Test execution
b) Developing
c) Product release
d) Defect finding

3) Which of the following is not a testing principle? Show Answer

The correct Answer is 'C'
a) Testing shows presence of defects
b) Exhaustive testing is impossible
c) Failure clustering
d) Early testing

4) To overcome a “pesticide paradox”, test cases need to be regularly what? Show Answer

The correct Answer is 'C'
a) Deleted
b) Executed
c) Reviewed
d) Released

5) Comparing actual results with expected results is associated with what? Show Answer

The correct Answer is 'C'
a) Evaluating Exit Criteria and Reporting
b) Test Closure Activities
 c) Test Implementation and Execution
d) Test Analysis and Design

6) Independent testing often makes the tester more effective at finding what? Show Answer

The correct Answer is 'C'
a) Requirement design issues
 b) Working as a team
c) Defects and failures
d) Completing tests on time

7) Which of the following has the lowest level of independence? Show Answer

The correct Answer is 'B'
a) Tests designed by another person(s)
b) Tests reviewed by the person(s) who wrote the software under test
c) Tests designed by a person(s) from a different organizational group
d) Tests designed by a person(s) from a different organization or company

8) Which of the following is not considered to be under the ISTQB code of ethics? Show Answer

The correct Answer is 'B'
a) Judgement
b) Client and Customer
c) Profession
d) Public

9) Which of the following is not a valid Iterative-incremental development model? Show Answer

The correct Answer is 'C'
a) Prototyping
b) Rapid Application Development (RAD)
c) Agility development models
d) Rational Unified Process (RUP)

10) Integration testing tests interfaces between what? Show Answer

The correct Answer is 'B'
a) Lines of code
b) Components
c) Sectors
d) Projects

11) Which of the following is not part of the test basis for System testing? Show Answer

The correct Answer is 'C'
a) System and software requirement specification
b) Functional specification
 c) Acceptance tests
d) Risk analysis reports

12) Which testing should investigate functional and non-functional requirements of the system? Show Answer

The correct Answer is 'A'
a) System testing
b) Regression testing
c) Functional testing
d) User acceptance testing

13) Which type of testing has the goal of establishing confidence? Show Answer

The correct Answer is 'C'
a) Regression testing
b) Functional testing
c) Acceptance testing
d) System testing

14) Which testing of a new functional enhancement may come before system testing? Show Answer

The correct Answer is 'C'
a) Regression testing
b) Functional testing
c) Acceptance testing
d) System testing

15) Acceptance criteria should be defined when the parties what? to the contract. Show Answer

The correct Answer is 'C'
a) Disagree
b) Define
c) Agree
d) Create

16) Which testing is performed at the developing organization’s site? Show Answer

The correct Answer is 'C'
a) Operational Acceptance testing
b) User Acceptance testing
c) Alpha testing
d) Contract and Regulation Acceptance testing

17) Which testing considers the external behavior of the software? Show Answer

The correct Answer is 'B'
a) User Acceptance testing
 b) Functional testing
c) Acceptance testing
d) Regression testing

18) Which testing is done on an existing operational system, and is triggered by modifications? Show Answer

The correct Answer is 'B'
a) Load testing
b) Maintenance testing
c) Usability testing
d) Confirmation testing

19) Reviews, static analysis and dynamic testing are said to be what? Show Answer

The correct Answer is 'C'
a) Pointless
b) Ad-hoc
c) Complementary
d) Black-box techniques

20) Who is the writer or person with chief responsibility for the document(s) to be reviewed? Show Answer

The correct Answer is 'B'
a) A Moderator
b) An Author
c) A Manager
d) A Scribe

21) Which of the following is not a characteristic of an Technical review? Show Answer

The correct Answer is 'C'
a) May be performed as a peer review
b) Pre-meeting preparation by reviewers
c) Varies in usefulness depending on the reviewers
d) Optional use of checklists

22) Which of the following is not a characteristic of an Inspection review? Show Answer

The correct Answer is 'D'
a) Pre-meeting preparation
b) Inspection report including list of findings
c) Formal follow-up process
 d) May vary in practice from quite informal to very formal

23) Static analysis tools may produce a large number of what? Show Answer

The correct Answer is 'D'
a) Print-outs
b) Defects
c) Syntax errors
d) Warning messages

24) Which of the following is not a purpose of the test design technique? Show Answer

The correct Answer is 'C'
a) Identify test conditions
 b) Identify test cases
c) Identify test specifications
d) Identify test data

25) A state transition diagram displays all but which of the following? Show Answer

The correct Answer is 'C'
a) Transitions between states
b) Inputs or events that trigger state changes
 c) The fields
d) The actions which may result from the transitions

26) What describes the “process flows” through a system based on its actual likely use? Show Answer

The correct Answer is 'C'
a) A test case
b) A test plan
c) A use case
d) A test procedure

27) Which testing is concurrent test design, test execution, test logging and learning? Show Answer

The correct Answer is 'A'
a) Exploratory testing
b) Decision testing
c) Functional testing
d) Error guessing

28) Which testers see other and different defects, and are unbiased? Show Answer

The correct Answer is 'B'
a) Functional testers
b) Independent testers
c) Structural Testers
d) Contract Testers

29) Test planning is influenced by all but which of the following? Show Answer

The correct Answer is 'C'
a) Scope of testing
b) Objectives
c) Repeatability
d) Risks

30) The testing effort may depend on all but which of the following factors? Show Answer

The correct Answer is 'B'
a) Characteristics of the product
b) Characteristics of the test process
c) Characteristics of the development process
d) The outcome of testing

31) The purpose of what is to provide feedback and visibility about test activities? Show Answer

The correct Answer is 'A'
a) Test monitoring
b) Test planning
c) Test progress
d) Test cases

32) What are the risks that surround the project’s capability to deliver its objectives? Show Answer

The correct Answer is 'A'
a) Project risks
b) Product risks
c) Acceptable risks
d) Unacceptable risks

33) What are potential failure areas in the software or system are known as? Show Answer

The correct Answer is 'D'
a) Acceptable risks
b) Unacceptable risks
c) Project risks
 d) Product risks

34) In a risk-based approach, the identified risks can be used to all but which of the following? Show Answer

The correct Answer is 'D'
a) Determine the test techniques to be employed
b) Determine the extent of testing to be carried out
c) Determine whether any non-testing activities could be employed to reduce risk
d) Prioritize defects in an attempt to find the critical defects as early as possible

35) Which tool provides interfaces for executing tests? Show Answer

The correct Answer is 'C'
a) Requirements Management Tools
b) Incident Management Tools
c) Test Management Tools
d) Configuration Management Tools

36) Which of the following is not a risk of using a tool? Show Answer

The correct Answer is 'C'
a) Unrealistic expectations for the tool
b) Over-reliance on the tool
c) Over-use of the tool
d) Underestimating the effort required to maintain the test assets

37) Success factors for the deployment of the tool include all but which of the following? Show Answer

The correct Answer is 'B'
a) Monitoring tool use and benefits
b) Defining users environments
c) Providing support for the test team for a given tool
d) Gathering lessons learned from all teams

38) Which of the following does not relate to introducing a pilot project? Show Answer

The correct Answer is 'B'
a) Evaluate how the tool fits with existing processes
b) Learn more detail about the users of the tool
c) Decide on standard ways of using the tool
d) Assess whether the benefits will be achieved at reasonable cost

39) Which tools are used to generate test inputs or executable tests? Show Answer

The correct Answer is 'C'
a) Test Data Preparation Tools
b) Test Execution Tools
c) Test Design Tools
d) Test Comparators

40) Some tools clearly support one _______, others may support more than one ______. Show Answer

The correct Answer is 'A'
a) Activity
b) Operating system
c) Tester
d) License
