Practice Quiz 6

1) Defects occur because human beings are what? Show Answer

The correct Answer is 'C'
a) Stupid
b) Erratic
c) Fallible
d) Complex

2) Which of the following is not a testing principle? Show Answer

The correct Answer is 'A'
a) Exhaustive testing is possible
b) Testing shows presence of defects
c) Defect clustering
d) Early testing

3) The most visible part of testing is what? Show Answer

The correct Answer is 'C'
a) Test plan
b) Test case design
c) Test execution
d) Defect analysis

4) Documenting the acceptance of the system is associated with what? Show Answer

The correct Answer is 'C'
a) Evaluating Exit Criteria and Reporting
b) Test Analysis and Design
c) Test Closure Activities
d) Test Implementation and Execution

5) Which of the following has the lowest level of independence? Show Answer

The correct Answer is 'C'
a) Tests designed by another person(s)
b) Tests designed by a person(s) from a different organizational group
c) Tests designed by the person(s) who wrote the software under test
d) Tests designed by a person(s) from a different organization or company

6) Identifying failures during testing may be perceived as what? Show Answer

The correct Answer is 'C'
a) A bonus
b) An opportunity
c) A criticism
d) A positive

7) Testing is often seen as what type of activity? Show Answer

The correct Answer is 'B'
a) Constructive
b) Destructive
c) Over-long
d) Un-managed

8) The tester and test leader need good interpersonal skills to what? Show Answer

The correct Answer is 'B'
a) Dictate
b) Communicate
c) Find defects
d) Review documentation

9) A common V-model uses all but which of the following test levels? Show Answer

The correct Answer is 'C'
a) Component testing
b) Integration testing
c) Installer testing
d) System testing

10) What does RUP represent? Show Answer

The correct Answer is 'B'
a) Rational Unified Project
 b) Rational Unified Process
c) Rational Unified Progress
d) Rational Untied Progress

11) Which approach is to prepare and automate test cases before coding? Show Answer

The correct Answer is 'C'
a) Ad-hoc approach
b) Agile approach
c) Test-first approach
d) Test-last approach

12) The greater the scope of integration, the more difficult it becomes to isolate what? Show Answer

The correct Answer is 'A'
a) Failures
b) Requirements
c) Software
d) Systems

13) Which of the following is not part of the test basis for System testing? Show Answer

The correct Answer is 'C'
a) System and software requirement specification
b) Use cases
c) Functional tests
d) Risk analysis reports

14) Which of the following is not a test object for Acceptance testing? Show Answer

The correct Answer is 'C'
a) User procedures
b) Business processes on fully integrated system
c) Test processes
d) Operational and maintenance processes

15) Which testing is performed at the customers site? Show Answer

The correct Answer is 'C'
a) Operational Acceptance testing
b) User Acceptance testing
c) Beta testing
d) Alpha testing

16) Which type of testing investigates the functions relating to detection of threats? Show Answer

The correct Answer is 'C'
a) User Acceptance testing
 b) Acceptance testing
c) Security testing
d) Regression testing

17) Which testing is based on patches to correct vulnerabilities of the operating system? Show Answer

The correct Answer is 'B'
a) Load testing
b) Maintenance testing
c) Usability testing
d) Confirmation testing

18) Reviews can find omissions in requirements, which are unlikely to be found in which testing? Show Answer

The correct Answer is 'B'
a) Performance testing
b) Dynamic testing
c) Functional testing
d) Static testing

19) The formality of a review process is related to all but which of the following? Show Answer

The correct Answer is 'C'
a) Maturity of the development process
b) Any legal or regulatory requirements
c) Maturity of the test process
d) The need for an audit trail

20) A typical formal review will include all but which of the following roles? Show Answer

The correct Answer is 'B'
a) Scribe
b) Documentor
c) Moderator
d) Author

21) Which of the following is not a characteristic of an Walkthrough review? Show Answer

The correct Answer is 'C'
a) Meeting led by author
 b) May take the form of scenarios
c) Pre-meeting preparation by reviewers
d) Open-ended sessions

22) Walkthroughs, technical reviews and inspections can be performed within a what? Show Answer

The correct Answer is 'D'
a) A Quango
b) A Test team
c) A Focus group
d) A Peer group

23) As with _______, static analysis finds defects rather than failures. Show Answer

The correct Answer is 'D'
a) Black-box testing
b) Regression testing
c) White-box testing
d) Reviews

24) What is defined as an item or event that could be verified by one or more test cases? Show Answer

The correct Answer is 'C'
a) A test process
b) A test matrix
c) A test condition
d) A test case

25) The _________ specifies the sequence of actions for the execution of a test. Show Answer

The correct Answer is 'C'
a) Test plan
b) Test specification
c) Test procedure
d) Test case

26) Which technique is often considered as an extension of equivalence partitioning? Show Answer

The correct Answer is 'C'
a) State transition testing
 b) Functional testing
c) Boundary value analysis
d) Acceptance testing

27) What can be derived from use cases? Show Answer

The correct Answer is 'A'
a) Tests
b) Requirements
c) Design documents
d) Test plans

28) A commonly used experienced-based technique is what? Show Answer

The correct Answer is 'B'
a) Decision testing
b) Error guessing
c) Boundary value analysis
d) Functional testing

29) Which testers can verify assumptions people made during implementation of the system? Show Answer

The correct Answer is 'C'
a) Functional testers
b) Structural Testers
c) Independent testers
d) Contract Testers

30) Which of the following is not a typical tester task? Show Answer

The correct Answer is 'B'
a) Prepare and acquire test data
b) Approve test specifications
c) Review tests developed by others
d) Set up the test environment

31) Which of the following is a valid test estimation approach? Show Answer

The correct Answer is 'A'
a) The expert-based approach
 b) The business-based approach
c) The Black-box approach
d) The White-box approach

32) Typical test approaches include all but which of the following? Show Answer

The correct Answer is 'A'
a) Political approaches
 b) Model-based approaches
c) Methodical approaches
d) Dynamic and heuristic approaches

33) What is concerned with summarizing information about the testing endeavor? Show Answer

The correct Answer is 'D'
a) Defect descriptions
b) Test plans
c) Project plans
d) Test reporting

34) All but which of the following is related to project risk? Show Answer

The correct Answer is 'D'
a) Organizational factors
 b) Technical issues
c) Supplier issues
 d) Software that does not perform its intended functions

35) Which testing draws on the collective knowledge and insight of the project stakeholders? Show Answer

The correct Answer is 'C'
a) Functional testing
b) Regression testing
c) Risk-based testing
d) Acceptance testing

36) Tools can be classified based by purpose on all but which of the following? Show Answer

The correct Answer is 'C'
a) Commercial
b) Free
c) Closed-source
d) shareware

37) Which test tool determines the differences between files, databases or test results? Show Answer

The correct Answer is 'B'
a) Test Data Preparation Tools
b) Test Comparators
c) Coverage Measurement Tools
d) Test Execution Tools

38) Which of the following is not a risk of using a tool? Show Answer

The correct Answer is 'B'
a) Unrealistic expectations for the tool
b) Risk of developing company going out of business
c) Over-reliance on the tool
 d) Underestimating the effort required to maintain the test assets

39) Which of the following is not a consideration in selecting a tool for an organization? Show Answer

The correct Answer is 'C'
a) Assessment of organizational maturity
b) Evaluation against clear requirements
c) Estimation of a cost-effect ratio
d) Evaluation of training needs

40) Which tools find defects that are evident only when software is executing? Show Answer

The correct Answer is 'A'
a) Dynamic Analysis Tools
b) Modeling Tools
c) Test Management Tools
d) Static Analysis Tools
