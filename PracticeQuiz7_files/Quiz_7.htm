Practice Quiz 7

1) Which of the following is not an environmental conditions? Show Answer

The correct Answer is 'C'
a) Radiation
b) Magnetism
c) Hardware
d) Pollution

2) Testing should provide sufficient information to who? Show Answer

The correct Answer is 'A'
a) Stakeholders
b) Customers
c) Rival companies
d) Developers

3) Which of the following is not a testing principle? Show Answer

The correct Answer is 'C'
a) Early testing
b) Defect clustering
c) Testing shows presence of mistakes
d) Exhaustive testing is impossible

4) Finding and fixing defects does not help if the system built is what? Show Answer

The correct Answer is 'C'
a) Released
b) Undesirable
c) Unusable
d) Revised

5) Verifying that the test environment has been set up correctly is associated with what? Show Answer

The correct Answer is 'C'
a) Evaluating Exit Criteria and Reporting
b) Test Closure Activities
 c) Test Implementation and Execution
d) Test Analysis and Design

6) Analyzing lessons learned to determine changes is associated with what? Show Answer

The correct Answer is 'C'
a) Evaluating Exit Criteria and Reporting
b) Test Analysis and Design
 c) Test Closure Activities
d) Test Implementation and Execution

7) People and projects are driven by what? Show Answer

The correct Answer is 'B'
a) Power
b) Objectives
c) Environment
d) Training

8) The ISTQB recognizes which of the following codes of ethics? Show Answer

The correct Answer is 'B'
a) EIII and ACM
b) ACM and IEEE
c) ISEB and IEEE
d) ABC DL

9) A common V-model uses all but which of the following test levels? Show Answer

The correct Answer is 'C'
a) Component testing
b) Integration testing
c) Functional testing
d) System testing

10) Iterative-incremental development includes all but which of the following? Show Answer

The correct Answer is 'B'
a) Establishing requirements
 b) Building and releasing a system
c) Designing
d) Done as a series of shorter development cycles

11) What does RAD represent? Show Answer

The correct Answer is 'C'
a) Release Application Development
b) Rapid Access Development
 c) Rapid Application Development
d) Rapid Application Deployment

12) Which type of development prepares and automates test cases before coding? Show Answer

The correct Answer is 'A'
a) Test-driven development
 b) Ad-hoc development
c) Agile development
d) Test-last development

13) Which testing is concerned with the behavior of a whole system/product? Show Answer

The correct Answer is 'C'
a) Regression testing
b) Functional testing
c) System testing
 d) User acceptance testing

14) Finding defects is not the main focus in which type of testing? Show Answer

The correct Answer is 'C'
a) Regression testing
b) Functional testing
c) Acceptance testing
d) System testing

15) Which testing is performed by potential customers? Show Answer

The correct Answer is 'C'
a) Operational Acceptance testing
b) User Acceptance testing
c) Beta testing
d) Alpha testing

16) Which of the following is not related to non-functional testing? Show Answer

The correct Answer is 'C'
a) Performance testing
b) Load testing
c) Confirmation testing
 d) Usability testing

17) Once deployed, a software system is often in service for how long? Show Answer

The correct Answer is 'B'
a) Days or weeks
b) Years or decades
c) Months or years
 d) Infinity

18) What are a way of testing software work products? Show Answer

The correct Answer is 'B'
a) Checking
b) Reviews
c) Planning
d) Releasing

19) The way a review is carried out depends on the agreed what? Show Answer

The correct Answer is 'C'
a) Location
b) Attendees
c) Objectives
d) Timescale

20) A typical formal review will include all but which of the following roles? Show Answer

The correct Answer is 'B'
a) Manager
b) Tester
c) Moderator
d) Author

21) Which of the following is not a characteristic of an Informal review? Show Answer

The correct Answer is 'C'
a) No formal process
b) Varies in usefulness depending on the reviewers
c) Optional use of checklists
d) Inexpensive way to get some benefit

22) Which of the following is not a success factor for a review? Show Answer

The correct Answer is 'D'
a) Each review has clear predefined objectives
b) The right people for the review objectives are involved
c) Defects found are welcomed and expressed objectively
d) There is an emphasis on process elimination

23) Which of the following is not a typical defect discovered by static analysis? Show Answer

The correct Answer is 'D'
a) Variables that are not used or are improperly declared
b) Referencing a variable with an undefined value
c) Syntax violations of code and software models
d) Security violations

24) Expected results should include all but which of the following? Show Answer

The correct Answer is 'C'
a) Include outputs
b) Changes to data and states
c) Changes to inputs and states
d) Any other consequences of the test

25) Equivalence partitions can be identified for all but which of the following? Show Answer

The correct Answer is 'C'
a) Outputs
b) Time-related values
c) External values
 d) Interfacing components

26) Which testing is widely used within the embedded software industry? Show Answer

The correct Answer is 'C'
a) Decision table testing
 b) Boundary value analysis
c) State transition testing
d) Equivalence Partitioning

27) Which technique derives test cases to execute specific statements? Show Answer

The correct Answer is 'A'
a) Statement testing
b) Functional testing
c) Decision test
 d) Boundary value analysis

28) 100% decision coverage guarantees 100% what? Show Answer

The correct Answer is 'B'
a) Decision coverage
b) Statement coverage
c) Boundary value analysis
d) Branch coverage

29) Which testers may make Developers lose a sense of responsibility for quality? Show Answer

The correct Answer is 'C'
a) Functional testers
b) Structural Testers
c) Independent testers
d) Contract Testers

30) Test planning is influenced by all but which of the following? Show Answer

The correct Answer is 'B'
a) Scope of testing
b) Availability of requirements
c) Risks
d) Criticality

31) Typical entry criteria may cover all but which of the following? Show Answer

The correct Answer is 'A'
a) Test date availability
 b) Test environment availability and readiness
 c) Test tool readiness in the test environment
 d) Testable code availability

32) The selected approach depends on all but which of the following? Show Answer

The correct Answer is 'A'
a) The nature of the test environment
b) The technology
c) The test objectives
d) Hazards and safety

33) Risk management activities provide a disciplined approach to do all but which of the following? Show Answer

The correct Answer is 'D'
a) Assess what can go wrong
 b) Determine what risks are important to deal with
c) Implement actions to deal with those risks
 d) Assess what has gone right so far

34) The consequence of intrusive tools is called what? Show Answer

The correct Answer is 'D'
a) Test effect
b) Code effect
c) Cost effect
d) Probe effect

35) Which tools store and manage incident reports? Show Answer

The correct Answer is 'C'
a) Configuration Management Tools
b) Test Management Tools
c) Incident Management Tools
d) Requirements Management Tools

36) Which tools enable tests to be executed automatically? Show Answer

The correct Answer is 'C'
a) Test Data Preparation Tools
b) Coverage Measurement Tools
c) Test Execution Tools
d) Test Comparators

37) Which of the following is not a risk of using a tool? Show Answer

The correct Answer is 'B'
a) Unrealistic expectations for the tool
b) Reflecting on relationships and integration issues
c) Over-reliance on the tool
 d) Underestimating the effort required to maintain the test assets

38) Success factors for the deployment of the tool include all but which of the following? Show Answer

The correct Answer is 'B'
a) Monitoring tool use and benefits
b) Gathering lessons learned from developers
c) Providing support for the test team for a given tool
d) Defining usage guidelines

39) Which tools store requirement statements? Show Answer

The correct Answer is 'C'
a) Incident Management Tools
 b) Configuration Management Tools
c) Requirements Management Tools
d) Test Management Tools

40) Details of an incident report may include all but which of the following? Show Answer

The correct Answer is 'A'
a) Conclusions, reviews and appendices
b) Expected and actual results
c) Scope or degree of impact on stakeholder(s) interests
d) Urgency/priority to fix
