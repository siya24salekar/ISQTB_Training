Practice Quiz 8

1) Rigorous testing can help to reduce what? Show Answer

The correct Answer is 'C'
a) Staff retention
b) Product sales
c) Risk
d) Potential

2) Dynamic testing can show failures that are caused by what? Show Answer

The correct Answer is 'A'
a) Defects
b) Developers
c) Testers
d) Faults

3) Which of the following is not a testing principle? Show Answer

The correct Answer is 'C'
a) Pesticide paradox
b) Testing is context dependent
c) Absence-of-failures fallacy
d) Exhaustive testing is impossible

4) Test plans should include time to be spent on all but which of the following? Show Answer

The correct Answer is 'C'
a) Designing test cases
 b) Preparing for execution
c) Evaluating test process
d) Planning the tests

5) Finalizing and archiving testware have been delivered is associated with what? Show Answer

The correct Answer is 'C'
a) Evaluating Exit Criteria and Reporting
b) Test Analysis and Design
c) Test Closure Activities
d) Test Implementation and Execution

6) Which of the following is not a valid level of independence? Show Answer

The correct Answer is 'C'
a) Tests designed by a person(s) from a different organizational group
b) Tests designed by a person(s) from a different organization or company
c) Tests designed by another computer(s)
d) Tests designed by the person(s) who wrote the software under test

7) The ISTQB recognizes which of the following codes of ethics? Show Answer

The correct Answer is 'B'
a) T4S and ACM
b) ACM and IEEE
c) ISEB and IEEE
d) ABC DL

8) Which of the following is not a valid software work product? Show Answer

The correct Answer is 'B'
a) Business scenarios or use cases
b) Time-tracking schedule
c) Requirements specifications
d) Design documents and code

9) The Test Basis consists of all, but which of the following? Show Answer

The correct Answer is 'C'
a) Component requirements
 b) Detailed design
c) Test cases
 d) Code

10) One approach to component testing is to prepare and automate test cases before what? Show Answer

The correct Answer is 'B'
a) Acceptance testing
b) Coding
c) Project completion
d) Product release

11) Systematic integration strategies may include all but which of the following? Show Answer

The correct Answer is 'C'
a) System architecture
b) Functional tasks
c) Development practices
d) Transaction processing sequences

12) Which of the following is not a test object for Acceptance testing? Show Answer

The correct Answer is 'A'
a) Techniques
b) User procedures
c) Forms
d) Reports

13) Which testing may assess the system’s readiness for deployment and use? Show Answer

The correct Answer is 'C'
a) Regression testing
b) Functional testing
c) Acceptance testing
d) System testing

14) Factory acceptance testing is another term for what? Show Answer

The correct Answer is 'C'
a) Operational Acceptance testing
b) User Acceptance testing
 c) Alpha & Beta testing
d) Regression testing

15) Non-functional testing may be performed at which level? Show Answer

The correct Answer is 'C'
a) Requirements review
b) Code Development
c) All Levels
 d) Functional testing

16) Regression testing may be performed where? Show Answer

The correct Answer is 'C'
a) Requirements review
b) Code Development
c) All levels
 d) Functional testing

17) Which testing techniques relies on the manual examination and automated analysis of the code? Show Answer

The correct Answer is 'B'
a) Performance testing
b) Static testing
c) Functional testing
d) Dynamic testing

18) Which of the following is not a valid review objective? Show Answer

The correct Answer is 'B'
a) Gain understanding
b) Fix defects
c) Educate testers and new team members
d) Discussion and decision by consensus

19) Who decides on the execution of reviews, allocates time in project schedules? Show Answer

The correct Answer is 'C'
a) A Moderator
b) An Author
c) A Manager
d) A Scribe

20) Which of the following is not a characteristic of an Walkthrough review? Show Answer

The correct Answer is 'B'
a) Meeting led by author
 b) Varies in usefulness depending on the reviewers
c) May take the form of scenarios
d) Open-ended sessions

21) Which of the following is not a characteristic of an Technical review? Show Answer

The correct Answer is 'C'
a) May be performed as a peer review
b) Pre-meeting preparation by reviewers
c) No formal process
d) Optional use of checklists

22) What can locate defects that are hard to find in dynamic testing? Show Answer

The correct Answer is 'D'
a) Test case reviews
b) Black-box testing
c) White-box testing
d) Static analysis

23) Which of the following is not a typical defect discovered by static analysis? Show Answer

The correct Answer is 'D'
a) Inconsistent interfaces between modules and components
b) Variables that are not used or are improperly declared
c) Referencing a variable with an undefined value
d) Missing and erroneous testers

24) A test case consists of all but which of the following? Show Answer

The correct Answer is 'C'
a) Execution preconditions
 b) Execution postconditions
c) Set of output values
d) Expected results

25) Equivalence partitions can be identified for all but which of the following? Show Answer

The correct Answer is 'C'
a) Outputs
b) Internal values
c) Inputs
d) Time-related values

26) Decision table input conditions and actions are most often stated as what? Show Answer

The correct Answer is 'C'
a) On or Off
b) Yes or No
c) True or False
d) Up or Down

27) What can help uncover integration defects? Show Answer

The correct Answer is 'A'
a) A use case
b) A decision table
c) Boundary value analysis
 d) A test procedure

28) Decision coverage is stronger than what? Show Answer

The correct Answer is 'B'
a) Decision testing
b) Statement coverage
c) Boundary value analysis
d) Branch testing

29) The choice of which test techniques to use depends on a number of what? Show Answer

The correct Answer is 'C'
a) Projects
b) Inputs
c) Factors
d) Tests

30) Which testers may be seen as a bottleneck or blamed for delays in release? Show Answer

The correct Answer is 'B'
a) Functional testers
b) Independent testers
c) Structural Testers
d) Contract Testers

31) The role of the test leader may be performed by all but which of the following? Show Answer

The correct Answer is 'A'
a) Business analyst
b) Project manager
c) Development manager
d) Quality assurance manager

32) Testers for operational acceptance testing would typically be who? Show Answer

The correct Answer is 'A'
a) Operators
b) Testers
c) Managers
d) Developers

33) Typical entry criteria may cover all but which of the following? Show Answer

The correct Answer is 'D'
a) Test data availability
b) Test environment availability and readiness
 c) Testable code availability
d) Test tool readiness in the customers own environment

34) The testing effort may depend on all but which of the following factors? Show Answer

The correct Answer is 'D'
a) Characteristics of the product
b) Characteristics of the development process
 c) The outcome of testing
d) The outcome of the developing

35) Common test metrics include all but which of the following? Show Answer

The correct Answer is 'C'
a) Percentage of work done in test case preparation
b) Test case execution
c) Dates of development milestones
d) Test coverage of requirements

36) Risk management activities provide a disciplined approach to do all but which of the following? Show Answer

The correct Answer is 'C'
a) Assess what can go wrong
 b) Determine what risks are important to deal with
c) Determine which testers are important to use
 d) Implement actions to deal with those risks

37) Which tool can affect the actual outcome of the test? Show Answer

The correct Answer is 'B'
a) Automated
b) Intrusive
c) Regression
d) Semi-automated

38) Which tools measure the percentage of code structures that have been exercised? Show Answer

The correct Answer is 'B'
a) Test Data Preparation Tools
b) Coverage Measurement Tools
c) Test Execution Tools
d) Test Comparators

39) Which of the following is not a risk of using a tool? Show Answer

The correct Answer is 'C'
a) Unrealistic expectations for the tool
b) Over-reliance on the tool
c) Underestimating the delivery time of the tool
d) Underestimating the effort required to maintain the test assets

40) In a __________ testing approach, the spreadsheet contains keywords describing the actions. Show Answer

The correct Answer is 'A'
a) Keyword-driven
b) Test-driven
c) Requirement-driven
 d) Data-driven
