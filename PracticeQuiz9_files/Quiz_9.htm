Practice Quiz 9

1) Software testing may also be required to meet what? Show Answer

The correct Answer is 'C'
a) The customers
b) Stakeholders
c) Legal Requirements
d) Suggestions

2) Deciding how much testing is enough should take account of the level what? Show Answer

The correct Answer is 'A'
a) Risk
b) Knowledge
c) Defects
d) Timescales

3) Which of the following is not a testing principle? Show Answer

The correct Answer is 'C'
a) Exhaustive testing is impossible
b) Testing is context dependent
c) Testing is context independent
d) Pesticide paradox

4) Although logically sequential, the activities in the fundamental test process may what? Show Answer

The correct Answer is 'C'
a) Cross-reference
b) Repeat
c) Overlap
d) Duplicate

5) Checking which planned deliverables have been delivered is associated with what? Show Answer

The correct Answer is 'C'
a) Evaluating Exit Criteria and Reporting
b) Test Analysis and Design
c) Test Closure Activities
d) Test Implementation and Execution

6) Which of the following has the highest level of independence? Show Answer

The correct Answer is 'C'
a) Tests designed by another person(s)
b) Tests designed by a person(s) from a different organizational group
c) Tests designed by a person(s) from a different organization or company
d) Tests designed by the person(s) who wrote the software under test

7) Which of the following has the lowest level of independence? Show Answer

The correct Answer is 'B'
a) Tests designed by another person(s)
b) Tests reviewed by a person(s) from a different organizational group
c) Tests designed by a person(s) from a different organization or company
d) Tests designed by the person(s) who wrote the software under test

8) Which of the following is not considered to be under the ISTQB code of ethics? Show Answer

The correct Answer is 'B'
a) Judgement
b) Project
c) Profession
d) Public

9) Which of the following is a valid Software life cycle process? Show Answer

The correct Answer is 'C'
a) IEEE/T4S 12207
b) IEEE/IEC 72207
c) IEEE/IEC 12207
d) JJS-C/IEC 12207

10) Structural testing is typically performed by which of the following? Show Answer

The correct Answer is 'B'
a) Robustness testing
b) Regression testing
c) Component testing
d) Functional testing

11) During Integration testing, which non-functional testing can also be used? Show Answer

The correct Answer is 'C'
a) White-box
b) Structural
c) Performance
d) Ad-hoc

12) System testing may include tests based on risks and/or all but which of the following? Show Answer

The correct Answer is 'A'
a) Design documents
b) Requirements specifications
c) Business processes
d) Use cases

13) The goal in acceptance testing is to establish what in the system? Show Answer

The correct Answer is 'C'
a) Completeness
b) Performance
c) Confidence
d) Results

14) A test type is focused on a particular what? Show Answer

The correct Answer is 'C'
a) Test plan
b) Test result
c) Test objective
d) Test team

15) Structural testing may be performed at which level? Show Answer

The correct Answer is 'C'
a) Requirements review
b) Code Development
c) All levels
 d) Functional testing

16) Tests should be repeatable if they are to be used for what? Show Answer

The correct Answer is 'C'
a) Performance testing
b) Load testing
c) Confirmation testing
 d) Usability testing

17) The scope of maintenance testing is related to all but which of the following? Show Answer

The correct Answer is 'B'
a) Risk of the change
b) The size of the original test plan
c) The size of the existing system
d) Size of the change

18) Benefits of reviews include all but which of the following? Show Answer

The correct Answer is 'B'
a) Development productivity improvements
b) Decreased communication
 c) Early defect detection and correction
d) Reduced development timescales

19) Which of the following is not a valid formal review activity? Show Answer

The correct Answer is 'C'
a) Kick-off
b) Planning
c) Team preparation
d) Individual preparation

20) Which of the following is not a characteristic of an Informal review? Show Answer

The correct Answer is 'B'
a) No formal process
b) Optional scribe
c) Varies in usefulness depending on the reviewers
d) Inexpensive way to get some benefit

21) Which of the following is not a success factor for a review? Show Answer

The correct Answer is 'C'
a) Each review has clear predefined objectives
b) The right people for the review objectives are involved
c) Training is given in process improvement
d) Defects found are welcomed and expressed objectively

22) Which of the following is not a value of static analysis? Show Answer

The correct Answer is 'D'
a) Identification of defects not easily found by dynamic testing
b) Detecting dependencies and inconsistencies in software models such as links
c) Improved maintainability of code and design
d) Late detection of defects prior to test execution

23) A test case consists of all but which of the following? Show Answer

The correct Answer is 'D'
a) Set of input values
b) Execution preconditions
c) Expected results
d) Execution results

24) Equivalence partitions can be identified for all but which of the following? Show Answer

The correct Answer is 'C'
a) Outputs
b) Internal values
c) Interfacing components
d) Time-related values

25) A State transition test can be used for all but which of the following? Show Answer

The correct Answer is 'C'
a) Typical sequence of states
b) Cover every state
c) Exercise specific sequences of actions
d) Exercise every transition

26) A use case usually has a _______ scenario and alternative paths. Show Answer

The correct Answer is 'C'
a) Typical
b) Regression
c) Mainstream
d) Random

27) Exploratory testing involves all but which of the following? Show Answer

The correct Answer is 'A'
a) Test distribution
b) Test design
c) Test execution
d) Learning

28) Which testers can be seen to be in isolation from the development team? Show Answer

The correct Answer is 'B'
a) Functional testers
b) Independent testers
c) Structural Testers
d) Contract Testers

29) Typically testers at the component and integration level would be who? Show Answer

The correct Answer is 'C'
a) Testers
b) Managers
c) Developers
d) Business analysts

30) Typical entry criteria may cover all but which of the following? Show Answer

The correct Answer is 'B'
a) Test environment availability and readiness
b) Test process availability
 c) Test tool readiness in the test environment
 d) Testable code availability

31) The Characteristics of the product include all but which of the following? Show Answer

The correct Answer is 'A'
a) The requirements for test specification
b) The size of the product
 c) The complexity of the problem domain
d) The requirements for reliability and security

32) Typical test approaches include all but which of the following? Show Answer

The correct Answer is 'A'
a) Methodist approaches
 b) Analytical approaches
c) Model-based approaches
d) Dynamic and heuristic approaches

33) Which of the following is not an organizational factor relating to project risk? Show Answer

The correct Answer is 'D'
a) Skill, training and staff shortages
b) Personnel issues
c) Improper attitude toward or expectations of testing
d) Improper political attitude

34) Risk management activities provide a disciplined approach to do all but which of the following? Show Answer

The correct Answer is 'D'
a) Assess what can go wrong
b) Determine what risks are important to deal with
c) Implement actions to deal with those risks
 d) Assign risk to each testers to deal with

35) Which tools are necessary for storage and version management of testware? Show Answer

The correct Answer is 'C'
a) Configuration Management Tools
b) Test Management Tools
c) Configuration Management Tools
d) Requirements Management Tools

36) Which of the following is not a potential benefits of using tools? Show Answer

The correct Answer is 'C'
a) Repetitive work is reduced
b) Greater consistency and repeatability
c) Objective testing
d) Ease of access to information about tests or testing

37) Details of an incident report may include all but which of the following? Show Answer

The correct Answer is 'B'
a) Expected and actual results
b) Urgency/priority to test
c) Software or system life cycle process in which the incident was observed
 d) Urgency/priority to fix

38) Details of an incident report may include all but which of the following? Show Answer

The correct Answer is 'B'
a) Expected and actual results
b) Scope or degree of impact on testers interests
c) Software or system life cycle process in which the incident was observed
d) Urgency/priority to fix

39) What can be used measure the quality of software? Show Answer

The correct Answer is 'C'
a) Sales figures
b) Production feedback
c) Testing
d) Developing

40) Development testing includes but which of the following? Show Answer

The correct Answer is 'A'
a) Functional testing
b) Component testing
c) Integration testing
d) System testing
